\# Automated Grow-Bench Prototype



\## Overview



The Automated Grow-Bench Prototype is an Arduino Uno-based smart irrigation system that automatically controls watering using soil moisture measurements. The system continuously monitors soil moisture and activates a relay-controlled water pump when the soil becomes dry. Irrigation stops when sufficient moisture is reached, ensuring efficient water usage.



The prototype supports both automatic and manual operation. A push button provides manual override, while serial commands (`STATUS`, `FORCE\_ON`, and `FORCE\_OFF`) allow monitoring and control through the Serial Monitor. The project demonstrates a simple, reliable, and cost-effective solution for smart irrigation and agricultural automation.
